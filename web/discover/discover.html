Chat

1:1 Chat, Group Chat and Multi-point chat

The PubNub SDK is the most flexible and customizable way to add chat features like direct and group messaging, typing indicators and message reactions. PubNub offers a highly-scalable architecture with low-latency delivery

Virtual Events

Engage and react during high-occupancy live events in real time

Virtual events chat gives users the ability to engage with high-occupancy events such as watching sports, concerts, tutorials and more.

People Nearby (Geolocation)

PubNub is ideal for sharing Geolocation data in real-time

Connect users and devices worldwide, sharing location data privately and securely to enhance any widely distributed solution

Public Whiteboard (Collaboration)

Collaborate with colleagues, friends and strangers in real-time

With PubNub, you can create a real-time collaborative demo in the web using PubNub's JavaScript SDK to update users in real-time with drawings, name changes, and mouse positions.

Find my Luggage (IoT)

PubNub's infrastructure is ideal for IoT solutions with thousands of devices spread across a large geographical area

PubNub can help you manage your distributed portfolio of IoT devices, either built into an existing product you produce or by integrating with 3rd party sensors. Using PubNub you can communicate bidirectionally with global assets in real time with minimum latency.

Real-Time Streaming

PubNub's real-time streaming APIs allow you to generate, process and deliver streaming data to any number of subscribers

PubNub's Data Stream Network handles keeping both publishers and subscribers securely connected and ensuring that every piece of data is generally available in real-time, so scale is never an issue. Streamed data can be seamlessly aggregated into a single source of truth from which you can trigger actions or perform analytics.
